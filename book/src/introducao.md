# Introdução
Este é um projeto de API para o processo seletivo da empresa [Power of Data](https://www.powerofdata.ai/) para a vaga de Desenvolvedor Backend Junior.

## Case Técnico

**Contexto:**

Venha fazer parte da equipe inovadora da PowerOfData e participe na criação de uma plataforma incrível dedicada aos fãs de Star Wars! Queremos que você demonstre suas habilidades de desenvolvedor explorando a API do Star Wars, e criando funcionalidades que proporcionem uma experiência rica e interativa para os usuários.

**Objetivo:**

Construir e disponibilizar uma plataforma que visa oferecer uma experiência envolvente, onde os usuários podem explorar informações detalhadas sobre personagens, planetas, naves e filmes da saga. Para isso, necessitamos da criação de uma API capaz de capturar as necessidades dos usuários e responder perguntas básicas sobre os filmes, personagens, planetas e naves.

**Critérios de aceite:**

- Utilizar o ambiente GCP para construção da solução. Considere utilizar os serviços Cloud Functions e API Gateway/Apigee.


- Utilize Python como linguagem principal do projeto.


- Obtenha os dados a partir da API do Star Wars (https://swapi.dev/).


- Crie um endpoint que permita ao usuário consultar diferentes informações baseadas nos dados da API do Star Wars.


- Permita aos usuários interagirem com o sistema utilizando filtros com base em parâmetros específicos.

**Critérios de avaliação:**

- Capacidade de entender e aplicar regras de negócio.


- Implementação das funcionalidades sugeridas.


- Boas práticas de desenvolvimento.


- Lógica de programação.


- Capacidade de agregar valor e propor novas ideias e melhorias.

**Entrega e Avaliação:**

- Etapa 1: Você terá 7 dias, até 05/02, para realizar o desafio e compartilhar seu código em um repositório Git (GitHub ou similar), juntamente com a documentação técnica para análise.


- Etapa 2: Avaliação da documentação enviada pelo time da PowerOfData (etapa eliminatória).


- Etapa 3: Apresentação do projeto (20 minutos) para o time da PowerOfData, seguida de entrevista — caso você seja selecionado na etapa 2.

**Entregue mais valor com:**

- Implementação de funcionalidades extras que possam trazer complexidade ao código.


- Capacidade de ordenar os resultados de consultas.


- Opção de consultar informações correlacionadas, como personagens em um filme específico.


- Funcionalidades extras que demonstrem sua habilidade em lidar com dados complexos e proporcionem uma experiência aprimorada para o usuário.


- Inclusão de testes unitários.


- Métodos de autenticação.


- Apresentação de desenho de arquitetura técnica detalhada.
